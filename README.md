🧠 AI Memory Map

An AI-powered application that transforms study materials into interactive knowledge graphs.

The project extracts concepts and relationships from notes or PDF documents and visualizes them as a memory map, helping students understand and connect topics more effectively.

🚀 Features
- Read text files
- Extract important concepts using NLP
- Identify relationships between concepts
- Generate interactive knowledge graphs
- Extract text from PDF documents

 🛠️ Tech Stack
- Python
- spaCy
- PyMuPDF
- PyVis
- NetworkX
